## 写在前面

只是突然就有兴趣了，就开了这个坑，但是这个坑能不能填上就不清楚了

很喜欢克里斯蒂·阿加莎的《无人生还》，就想着用这个做为模板写一个故事

使用的规则可能连村规也算不上，毕竟我自己也没怎么玩过

就是想给自己找点事做，顺道练习练习叙事结构等等手法

如果顺利的话还会尝试将这个故事做成移动端的游戏之类的

**写在2021年4月10日**，实在没什么时间更新这个部分，只能暂时暂停这个计划了，总感觉要写故事的话还是要有充足的时间，非常遗憾我没能一直坚持下去，期待有回归的一天
