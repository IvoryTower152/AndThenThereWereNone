## 写在前面

只是突然就有兴趣了，就开了这个坑，但是这个坑能不能填上就不清楚了

很喜欢克里斯蒂·阿加莎的《无人生还》，就想着用这个做为模板写一个故事

使用的规则可能连村规也算不上。。毕竟我自己也没怎么玩过

就是想给自己找点事做的，顺道练习练习叙事结构等等功底

当然，如果顺利的话还会尝试将这个故事做成移动端的游戏之类的

以上，就是到目前为止的想法

(ps：希望我能坚持下去，不要再让这个想法夭折了)
